# Sistema Check-in Hotel

Sistema de console desenvolvido em Java para gerenciamento de reservas e realização de check-in de hóspedes.

O projeto foi criado com foco em prática de Programação Orientada a Objetos (POO), arquitetura MVC e validações de regras de negócio.

---

## Funcionalidades

- Cadastro de reservas
- Listagem de reservas
- Realização de check-in
- Validação de pagamentos
- Controle de status da reserva
- Regras para clientes VIP
- Armazenamento em memória utilizando listas

---

## Conceitos aplicados

- Programação Orientada a Objetos
- MVC
- Interfaces
- Composição
- Validações de negócio
- Manipulação de listas e objetos

---

## Tecnologias utilizadas

- Java
- IntelliJ IDEA

---

## Como executar

1. Clone o repositório
2. Abra o projeto na IDE
3. Execute a classe `Principal.java`
